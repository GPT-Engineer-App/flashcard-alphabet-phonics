## Setup

```sh
git clone https://github.com/GPT-Engineer-App/flashcard-alphabet-phonics.git
cd flashcard-alphabet-phonics
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
